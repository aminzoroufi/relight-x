# Scene Notes

Create a scene named `ReLightXHighway.unity` and add:

- One `ReLightXScene` GameObject with `HighwaySceneManager` and `MQTTClientBridge`.
- A directional light and camera looking down the highway.
- Optional Canvas with UI Text elements wired to `DashboardUI`.
- Optional transparent zone objects driven by `ZoneVisualizer`.

The scripts can generate poles and lanes at runtime, so the scene can start nearly empty.
